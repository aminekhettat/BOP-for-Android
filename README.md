# BOP for Android (Back Office Player)

BOP (Back Office Player) est un lecteur audio spécialisé conçu pour l'étude instrumentale et la pratique intensive. Initialement développé pour Windows en Python, cette version Android native offre une expérience mobile fluide et entièrement accessible.

## Fonctionnalités Clés
- **Contrôle Indépendant** : Ajustez le tempo (vitesse) et le pitch (tonalité) indépendamment.
- **Bouclage A-B Précis** : Définissez des segments pour travailler des passages spécifiques.
- **Gestion de Segments** : Nommez, sauvegardez et organisez vos sections de travail.
- **Session de Pratique Progressive** : Augmentez automatiquement le tempo à chaque itération.
- **Accessibilité Totale** : Entièrement compatible avec TalkBack et optimisé pour les utilisateurs non-voyants.

## Installation
Le projet est en cours de développement. Pour compiler localement :
1. Clonez le dépôt.
2. Ouvrez avec Android Studio Jellyfish ou plus récent.
3. Synchronisez Gradle.
4. Lancez sur un émulateur ou un appareil réel.

## Engagement pour l'Accessibilité
BOP a été conçu avec une approche **Accessibility-First** :
- **Optimisation TalkBack** : Chaque bouton et curseur possède des labels sémantiques clairs et des indices d'état (ex: "Tempo à 80%").
- **Contrastes Élevés** : L'interface respecte les ratios de contraste WCAG 2.1 AAA.
- **Navigation Linéaire** : L'ordre de focus est logiquement structuré pour une navigation fluide au balayage.
- **Retour Haptique** : Des vibrations subtiles confirment les actions critiques (définition de boucle, fin de session).
